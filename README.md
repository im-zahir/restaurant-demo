# 🍽️ DineNext — Premium Restaurant Management System

A high-performance, **WhatsApp-native** restaurant solution built with **Next.js 15, Tailwind CSS 4, and Supabase**. This system features a luxurious user interface, dynamic menu management, and a robust reservation system.

## ✨ Key Features

-   **💎 Premium Design**: Dark-mode optimized, glassmorphic UI with smooth Framer Motion animations.
-   **📱 WhatsApp-Native Ordering**: Customers can browse the menu, add items to their cart, and send orders directly via WhatsApp.
-   **📅 Table Booking**: A built-in reservation system for seamless dining experiences.
-   **🛠️ Admin Dashboard**: Full CRUD interface for managing menu items, categories, and bookings.
-   **⚡ Live Database**: Real-time updates powered by Supabase.

## 🚀 Tech Stack

-   **Frontend**: Next.js 15 (App Router), React 19, TypeScript
-   **Styling**: Tailwind CSS 4, Lucide React (Icons), Framer Motion (Animations)
-   **Backend**: Supabase (Database, Auth, Storage)
-   **Platform**: Vercel

## ⚙️ Getting Started

### 1. Prerequisites
- Node.js (Latest)
- Supabase account

### 2. Installation
```bash
git clone https://github.com/im-zahir/restaurant-demo.git
cd restaurant-demo
npm install
```

### 3. Environment Variables
Create a `.env.local` file in the root directory:
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 4. Run Locally
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to see your demo!
